Configurando sudo para usuário comum no Debian 10 Buster

3 de julho, 2020 por Malkon Faria

Criando usuário comum

Somente um administrador do sistema(root) ou outro usuário com acesso privilegiado pode criar usuários no sistema. Para criar um usuário entre com: 

adduser nome_do_usuário

Exemplo:

adduser gustavo

Este usuário não possui acesso privilegiado a certos comandos e arquivos no sistema. Por exemplo, ele não pode criar outros usuários, não pode desligar o sistema, reiniciar etc. Para ganhar este acessos, ele deverá fazer parte do grupo de usuário denominado sudo.

Para fazer isso, logue com usuário root ou mude para este usuário. Para isso entre com:

su

Após isso, entre com o comando para colocar o usuário gustavo no grupo de usuários sudo, concedendo assim acessos especiais à este usuário no sistema. 

usermod -aG sudo gustavo
