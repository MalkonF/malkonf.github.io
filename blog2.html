Usando TCP Wrappers para permitir a conexão de hosts específicos

TCP Wrappers é um sistema de controle de acesso de rede baseado em host. Endereços ip's e nomes de hosts podem ser usados para permitir ou negar acesso a serviços de rede no seu servidor.

O TCP Wrappers primeiramente consulta o arquivo /etc/hosts.deny para ver quais hosts não podem acessar determinados serviços. Depois, consulta o arquivo /etc/hosts.allow para verificar se existe alguma regra que permite que determinados hosts se conectem a serviços específicos. Primeiro vamos configurar o arquivo /etc/hosts.deny para negar qualquer host de acessar o serviço SSH:

    sudo nano /etc/hosts.deny

Agora, vamos criar uma regra em /etc/hosts.allow para permitir apenas hosts específicos a usar o serviço SSH:

    sudo nano /etc/hosts.allow
